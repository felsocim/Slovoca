General functions

Changing display language

The user interface of Slovoca is multilingual. Choose your preferred language from the list of currently supported languages using the Language item in the main menu of the application.

Closing the application

To quit the Slovoca application, select Exit either from the toolbar or the main menu of the application. You can use Alt+F4 hotkey as well. If there are any unsaved changes made to your current vocabulary project (if any), you will be asked whether you want to save them, discard them or cancel the application's close.

Accessing help file

To access this help file, click on Help button on the toolbar or select Help > View Help from the main menu of the application. You can use F1 hotkey as well.

Accessing the Slovoca's website

To view the Slovoca's website in your default browser, click select Help > Slovoca Online from the main menu of the application.

View brief information about Slovoca

To show basic information about Slovoca, its author, your current version installed or its license, select Help > About Slovoca from the main menu of the application. You can use Alt+F1 hotkey as well.
